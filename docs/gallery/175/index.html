Фрагмент натюрморта с виноградом, грушей и яблоками.

Этот кусочек я доснимал уже не в студии, а дома, с использованием обыкновенной вспышки. Чтобы ограничить освещенную область, к вспышке был «пристроен» самодельный тубус из картона.

При съемке использовался объектив Зенитар-М 1.8/50 с двумя малыми удлиннительными кольцами.
