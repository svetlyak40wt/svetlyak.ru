Этот снимок сделан с помощью тройной мультиэкспозиции.

Сначала был снят контрастный сюжет с черными веточками на белом фоне, а затем на него наложены еще два с текстурами тканей.

Получилось похоже на рисунок. Довольно фактурно, на мой взгляд.

Картинка «сводилась» в Photoshop Elements, для каждого слоя был выставлен режим смешивания Screen.
