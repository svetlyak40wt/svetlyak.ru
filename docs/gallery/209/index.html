Этот храм расположен на территории кремля в городе Коломны.

К сожалению, точного его названия я не знаю.

С куполами этой церкви вышел небольшой казус — там на заднем плане видны купола Успенского собора, что и помешало мне правильно идентифицировать храм при поиске в базе на http://temples.ru.

Огромное спасибо хочется сказать тем людям, которые придумали, и сделали такую грандиозную подсветку всех архитектурных сооружений Коломнского кремля. Только ради этого стоит приехать туда вечером! Запасайтесь штативами и пленкой, для съемки с длинной выдержкой.

Этот кадр, по правде говоря, составлен из девяти отдельных снимков, каждый из которых снят «с рук» без какой-либо дополнительной подготовки. Так уж получилось. Для склейки использовалась бесплатная программа hugin.

Более подробное описание храма, смотрите на temples.ru.
