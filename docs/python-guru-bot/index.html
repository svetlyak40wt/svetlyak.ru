Портфолио кейс: Telegram бот для изучающих Python

Описание проекта

Цель заказчика - привлечь аудиторию интересующуюся программированием, и предложить им консультации с настоящим наставником в случае если нужна серьезная помощь.

Функции бота

  • Отвечать на вопросы ученика согласно заданному промпту и знаниями которыми обладает нейросеть.
  • В случае серьезных затруднений предлагать ученику устроить созвон с репетитором.
  • Создавать встречу в Google Календаре и отправлять ссылку ученику.

Особенности реализации

Бот собран на конструкторе Suvvy.
