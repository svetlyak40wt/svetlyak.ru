Мой Debian репозиторий

Август 10 2007 12:29
Метки:
debian, репозиторий, линукс, красный
(все метки)

Занявшись обработкой отпускных фотографий, я с сожалением заметил, что в Debian stable, и даже в Ubuntu Studio версия ufraw далеко не самая последняя. Это и стало последней каплей, переполнившей чашу моего терпения. Так я завел свой собственный debian репозиторий.

Те, кто пользуются Windows, MacOS или другими дистрибутивами Linux, могут дальше не читать :)

Что там есть

Пока я собрал для debian stable (etch) только несколько программ, зато самые свежие версии:

  • exiv2 0.15
  • ufraw 0.12
  • qtpfsgui 1.8.11
  • inkscape 0.45.1
  • gimp-lqr-plugin 0.2.1 — плагин для масштабирования с сохранением пропорций.
  • audacity 1.3.3 — программа для работы со звуком, backport из testing.

Как настроить

Для настройки, вам нужно добавить новый источник пакетов в файл /etc/apt/sources.list. Для этого, откройте его в своем любимом текстовом редакторе, например в vim:

linux:~# vim /etc/apt/sources.list

И добавьте такую строчку:

deb http://svetlyak.ru/debian etch main contrib non-free

Это говорит apt, что на сайте svetlyak.ru можно брать deb пакеты. Для того, чтобы что-либо установить из подключенного репозитария, нужно сначала обновить базу пакетов вот такой командой:

linux:~# apt-get update

Все пакеты имеют gpg подпись, и для того, чтобы apt при установке мог ее проверить, вам нужно скачать и установить мой публичный ключ:

linux:~# wget http://svetlyak.ru/svetlyak.asc  
linux:~# apt-key add svetlyak.asc

Все, теперь можно устанавливать ПО. Например с помощью apt-get:

linux:~# apt-get install ufraw qtpfsgui

Что еще

В ближайшее время, я наверное расширю список пакетов такими программами, как fspot 4.0, hugin 0.7 и, быть может, gimp 2.3. Кроме того, скоро ждите так же репозитарий для ubuntu.

Если есть какие-то определенные пожелания, то я могу и другой софт собрать. Только прошу учесть, что он должен иметь какое-либо отношение к фотографии и обработке изображений.
