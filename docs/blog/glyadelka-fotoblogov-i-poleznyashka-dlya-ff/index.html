Гляделка фотоблогов и полезняшка для FF

Май 20 2007 15:11
Метки:
собиралка
(все метки)

Буквально в несколько строк, хочу рассказать об одном довольно интересном сайте, и о полезном расширении к Firefox.

И так, наткнулся я недавно на сайт «VFXY Photos». Он представляет собой агрегатор фотоблогов. То есть, этот сайт автоматически получает новый контент с различных фотоблогов, и позволяет просмотривать его в одной ленте. Там есть фильтры, которые позволяют отобрать для просмотра только интересующие вас ресурсы. Какие же преимущества может дать вам этот замечательный сайт-агрегатор? Во-первых, он может стать для вас отличным онлайн ресурсом для просмотра обновлений сразу десятков фотоблогов. Во-вторых, если вы сами ведете фотоблог, то VFXY Photos, это еще одно место, где вы можете показать миру свои замечательные фотографии. Я уже зарегистрировался :).

Еще одной интересной находкой для меня, оказалось расширение к браузеру Firefox, которое называется «Exif Viewer». После его установки, в контестное меню картинок добавляется пункт «View Image Exif Data». Он позволяет просмотреть всю информацию об изображении, а раньше мне для этого приходилось каждый раз сохранять картинку на локальном диске и просматривать информацию из exif в отдельной программе.
