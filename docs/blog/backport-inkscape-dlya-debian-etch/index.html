Backport Inkscape для debian etch

Август 29 2007 12:53
Метки:
debian, репозиторий, линукс, красный
(все метки)

Новость, буквально одной строкой. Собрал последнюю стабильную версию Inkscape 0.45 для debian etch. Именно с его помощью я рисую иконки для сайта, но под ubuntu. Теперь можно воспользоваться всеми возможностями этого замечательного векторного редактора могут и пользователи debian stable. Достаточно лишь подключить мой репозитарий.

Список вкусностей:

  • Гауссово размывание.
  • Ускорение отрисовки и интерактивность.
  • Новый диалог История действий.
  • Добавлено много новых расширений эффектов.

Более полный список новых фич и их описание, ищите на linuxgraphics.ru.

Ставится просто:

sudo apt-get update
sudo apt-get install inkscape

Если вам понравилась эта статья, следите за будущими обновлениями, используя RSS или подписку на email рассылку.
