Определяем кол-во срабатываний затвора

Август 21 2007 17:42
Метки:
советы, линукс, unix, exif, красный
(все метки)

Знаете ли вы, что определить количество срабатываний затвора можно прочитав exif информацию из снимка.

Под unix системами сделать это можно очень просто, например с помощью команды exiftool:
exiftool -g dsc_4522.jpg | grep -i count

Или то же самое, но с помощью утилиты exiv2:
exiv2 -pv dsc_4522.jpg | grep -i count

Для моих RAW файлов, снятых на Nikon D50, эти команды выводят следующее:
Shutter Count : 4879
и
0x00a7 Nikon3 ShutterCount Long 1 4879

Кстати, если вы точно знаете, как называется exif тег, то можно использовать и такую команду:
exiftool -ShutterCount dsc_4522.jpg

Счастливые обладатели debian и ubuntu могут установить себе exiftool и exiv2 из репозитария:
sudo apt-get install libimage-exiftool-perl exiv2
