Метки:

(все метки)
LR

После долгого перерыва, я наконец-то собрался с силами и дописал плагин для загрузки фотографий из Adobe Lightroom в фотосервис Яндекса.

Внутри плагин претерпел значительные изменения, и работает теперь по тому же принципу, что и плагин massupload для Firefox, загружая картинки не целиком, а по частям.

Кроме внутренних изменний, в плагине так же появилась долгожданная загрузка снимков в альбомы.

Простые пользователи могут скачать новую версию плагина. Те же, кому интересно поучаствовать в разработке, могут забрать исходники проекта из git репозитория с помощью команды:

git clone http://git.svetlyak.ru/YaFotki.lrplugin/

На этом я с вами прощаюсь, но надеюсть что не надолго. Оставляйте свои фичереквесты в комментах к этому посту, или же на специальной странице плагинов для Lightroom.
