Портфолио кейс: Telegram бот для отслеживания платежей

Цели проекта

Цель заказчика - упростить отслеживание платежей за различные занятия детей и прочих семейных платежей.

Функции бота

  • Создавать регулярные платежи и напоминать о них.
  • Отслеживать просроченные платежи.
  • Вести информацию о платежах в системе трекинга задач Яндекс Трекер.
  • Давать возможность создать новые платежи по заранее созданным пресетам или в произвольной форме.

Особенности реализации

  • Бот запрограммирован на языке Common Lisp.
  • Развернут на облачном сервере заказчика.
  • Интегрирован с API Yandex Tracker.
