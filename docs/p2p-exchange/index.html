Портфолио кейс: Telegram бот для P2P-обмена USDT

Цель проекта

Реализовать Telegram-бот для P2P-обмена USDT с ручным управлением оператора, комиссией для продавца, ограничениями по количеству операций и процедурами верификации пользователей.

Основные функции

  • Создание и управление заявками на продажу USDT
  • Автоматический расчет комиссии и резервируемой суммы
  • Система верификации пользователей
  • Поиск и бронирование заявок покупателями
  • Ручное управление транзакциями через оператора
  • Система разрешения споров

Сценарии использования

Для продавца

  • Создание заявки с указанием суммы и курса
  • Перевод USDT на адрес обменника
  • Подтверждение получения оплаты
  • Верификация через криптокошелек

Для покупателя

  • Просмотр доступных заявок
  • Бронирование и оплата заявки
  • Получение USDT после подтверждения
  • Верификация через банковскую карту
